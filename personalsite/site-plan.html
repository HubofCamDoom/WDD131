Overview

Purpose

The purpose of this website is to create a convient player tool for a homebrew tabletop war game utilizing a building brick system.

Audience

Myself, tinkering enthusiasts, and other fans of tabletop wargamming.

Dynamic elements

For my dynamic website elements, I want to include a library of all of the base units from the game that the user can browse through on individual cards. Each unit card will have the relative game information and an image of the unit. I also want to have an army creator that would allow a player to put together a list of individual units. I believe this will involve a lot of conditional and website queries to implement. Maybe I will also add a way to add new units to the base game.
